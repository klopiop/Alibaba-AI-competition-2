作品说明：
本项目是参加阿里天池ESA Pages边缘开发大赛的作品，是一个基于边缘计算和人工智能的智能应用。作品利用阿里云边缘服务(ESA)的全球分布式节点，提供低延迟、高可用的AI服务体验。

主要功能特点：
1. 多语言支持：支持中英文界面切换
2. 智能对话：集成AI对话功能，提供智能问答服务
3. 中医咨询：提供传统中医健康咨询服务
4. 占卜预测：融合传统文化元素的占卜功能
5. 用户系统：完整的用户注册、登录和个人中心
6. 历史记录：保存用户交互历史，方便查看和管理

技术架构：
- 前端：Next.js + React + TypeScript
- 后端：Edge Functions + Prisma ORM
- 数据库：云数据库
- 部署：阿里云ESA边缘服务
- 国际化：next-intl多语言支持

作品创新点：
1. 利用边缘计算实现全球低延迟访问
2. 融合传统文化与现代AI技术
3. 提供个性化的智能咨询服务
4. 响应式设计，适配多种设备
